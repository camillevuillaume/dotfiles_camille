Theme: Tokyo Night.

https://github.com/user-attachments/assets/14624606-a76d-4328-a330-69f4edd05afe

## Features

- Display manager: [ly](https://github.com/fairyglade/ly)
- Compositor: [Hyprland](https://github.com/hyprwm/Hyprland) and [Niri](https://github.com/YaLTeR/niri)
- Terminal: [Kitty](https://github.com/kovidgoyal/kitty)
- Bash and zsh
- Lock screen: hyprlock
- Notifications: swaync
- Wallpapers: hyprpaper for Hyprland and swww for Niri, including script for changing the wallpaper depending on the time of the day
- Idle: hypridle for Hyprland and swayidle for Niri
- Bar: [Waybar](https://github.com/Alexays/Waybar) with Arch package status support, Impala for wifi, bluetui for Bluetooth and wiremix for sound
- [Fastfetch fork with animated gif support](https://github.com/Maybe4a6f7365/fastfetch-gif-support)
- Prompt: [Starship](https://starship.rs/)
- Neovim with various plugins
- Other tools like yazi, eza, ncspot, cava, btop...
- Supporting different configurations for two hosts (laptop and desktop)

## Usage

Download or clone, then [stow](https://www.gnu.org/software/stow/) what you need. You will probably need to change the host names for Hyprland and Niri.
